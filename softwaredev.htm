Milorad Software Development

Milorad is currently developing five softwares; GUMP, KUTE, Dum Dum Fait, SuppaTaxKard, and Gisnep-.

GUMP (GNU Useless Manipulation Program)

GUMP is Milorad's art program and Milorad's main focus currently. It is still a work-in-progress, so nothing is uploaded to Github.

KUTE (KDE Useless Text Editor)

KUTE is Milorad's text editor and a side-quest for development.

Dum Dum Fait

A card game adapted into video game form. Won't be made in a long time.

SuppaTaxKard

A mod for SuperTuxKart. Almost complete.
