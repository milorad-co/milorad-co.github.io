Software About

Milorad is currently developing five softwares; GUMP, WMWF, Dum Dum Fait, SupaTaxKard, and MI.

GUMP (GNU Useless Manipulation Program)

GUMP is Milorad's art program and Milorad's main focus currently.
GUMP Github page

GUMP is also available in your browser! It is very buggy and has no saving/loading functionality, but it works! You can use it here.

WMWF (WordyMcWordFace)

WMWF is Milorad's text editor and a side-quest for development.

Dum Dum Fait

A card game adapted into video game form. Won't be made in a long time.

SuppaTaxKard

A mod for SuperTuxKart. Almost complete.

MI

MI, which stands for Milorad Installer, is a commandline installation application which can install files from the latest release of a Github repository.
