Software Music Guides

Milorad is currently developing four (five-ish) softwares; MICA (-GM and -RS), Dum Dum Fait, SupaTaxKard, and MAIA.

MICA-GM (Milorad Image Creation Application in GameMaker)

MICA is Milorad's art program, written in GameMaker. Development is almost complete - there will be one more patch before development is entirely stopped to work on other projects.

Status: Impending Deprecation

MICA-GM GitHub Page

MICA-RS (Milorad Image Creation Application in Rust)

Since rust is a much faster programming language, and GameMaker isn't exactly suited for work on art programs, we have decided to begin work on a new version of MICA, which is much faster and written in Rust!

Status: Heavy development

MICA-RS GitHub Page

Dum Dum Fait

A card game adapted into video game form. Won't be made in a long time.

Status: Concept Only

SupaTaxKard

A mod for SuperTuxKart. Almost complete.

Status: Nearly Complete

MAIA (Milorad Automated Installation Application)

MAIA is a commandline installation application which can install files from the latest release of a GitHub repository.

Status: Heavy Development
