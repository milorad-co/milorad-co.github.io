MICA is Milorad's art program, written in GameMaker. Development is almost complete - there will be one more patch before development is entirely stopped to work on other projects.
MICA Github page

Current development progress: Complete

Since rust is a much faster programming language, and GameMaker isn't exactly suited for work on art programs, we have decided to begin work on a new version of MICA, which will be much faster and written in Rust!
