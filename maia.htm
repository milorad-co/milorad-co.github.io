MAIA

MAIA is Milorad's terminal installation utility. You can use it to install software packages easily off of GitHub or to install Milorad's software

Syantax and usage


To install go to here and download the latest release (note at the time of writing MAIA only supports Linux), and to install run
> chmod 775 maia
> sudo ./maia setup
Where the file was downloaded, then it will install itself and can be run with simply typing maia. Milorad software can be installed with:
> sudo maia install [software name]
For example, to install Mica you can run:
> sudo maia install mica
And Mica will be installed to /usr/bin and can be run like any other software.
For a help menu run:
> maia help
To install from GitHub repositories you can run:
> sudo maia install [user]/[repo]

Limitations and notes

We currently only support debian linux and releases have to be packaged in a specific way for MAIA to work and install them, if you have any issues please put an issue on the GitHub or if you want to be even more helpful try diving into the sphagetti code to fix it!
