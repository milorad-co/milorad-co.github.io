Maia

This is Milorad's terminal installation utility you can use it to install software packages easily off of github or to install Milorad's software

Syantax and usage


to install go to here and download the latest release (note at the time of writing this only supports linux), and to install run
#chmod 775 maia
#sudo ./maia setup
where the maia download file is, then it will install itself and can be run with simply typing maia. Milorad software can be installed with by typing
#sudo maia install [software name]
like for Mica you can type
#sudo maia install mica
and mica will be installed to /usr/bin and can be run like any other software.
for a help menu type
#maia help
To install from github repos you can type
#sudo maia install [user]/[repo]

Limitations and notes

We currently only support debian linux and releases have to be packaged in a specific way for MAIA to work and install them, if you have any issues please put an issue on the github or if you want to be even more helpful try diving into the sphagetti code to fix it!
